# User Form App

## Objective

To create a simple form using React, hooks & functional components that takes 3 inputs:

- User Name
- User Age
- User Email

**Where email has to be unique.**

### `Dependencies`

+ bootstrap: 5.2.3
+ react: 18.2.0
+ react-bootstrap: 2.6.0
+ react-dom: 18.2.0
+ react-scripts: 5.0.1
+ web-vitals: 2.1.4
